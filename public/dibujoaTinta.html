¡Bienvenido a Dibujo a Tinta!

Materiales Necesarios

# Materiales básicos

1. Tinta: Puedes elegir entre diferentes tipos de tinta, como tinta china, tinta acrílica, tinta a base de agua o tinta a base de aceite.

2. Pluma o pincel: Una pluma o pincel de alta calidad es esencial para crear líneas precisas y expresivas.

3. Papel: Elige un papel de alta calidad que sea adecuado para el tipo de tinta que estás utilizando. El papel debe ser lo suficientemente grueso para evitar que la tinta se filtre.

Materiales adicionales

# Materiales adicionales

1. Tintero: Un tintero es útil para almacenar y transportar la tinta.

2. Papel secante: Un papel secante puede ayudar a absorber el exceso de tinta y evitar que se corra.

3. Lápiz: Un lápiz puede ser útil para crear un borrador o esbozo antes de empezar a dibujar con tinta.

4. Goma de borrar: Una goma de borrar puede ser útil para corregir errores o eliminar marcas de lápiz.
